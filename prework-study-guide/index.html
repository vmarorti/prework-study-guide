HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The img element contains information about the images that are displayed on the webpage.
  • The a element stands for anchor and creates links to the same webpage or other webpages.
  • The section elements are semantic elements that define a section in a document that contains thematic content.
  • The section elements will allow us to organize the content on our page based on the topics we have.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside the element.
  • Declarations contain two important components: The CSS property we want to apply and the value of the property
  • To apply a rule to all the elements visible on the page, we use a special symbol, or *.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch name: creates a new branch and switches to it
  • git add -a: adds modifications in the current working branch to the staging area
  • -a: indicates that we want to add all changes

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • We can use conditional statements and loops to control the order in which code is executed, or if it is even executed at all.
  • An if statement is a way of introducing decision-making into our code.
  • An array is a single variable that is used to hold a group of data.
