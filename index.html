HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The img element contains information about the images that are displayed on the webpage.
  • The a element stands for anchor and creates links to the same webpage or other webpages.
  • The section elements are semantic elements that define a section in a document that contains thematic content.
  • The section elements will allow us to organize the content on our page based on the topics we have.
